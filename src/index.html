Easy to Use

NPM

Use NPM package manager to add modules to your project

GULP

Comes with an advanced GULP file for sourcing components

  • Copy Fonts, JS, & CSS from node modules
  • Copy HTML files from src folder
  • Automatically inject partials into HTML file
  • Automatically inject js & css files into
  • Automatically compile css
  • Automatically copy JS & use browserify to allow for using ES6 modules
  • Browsersync automatically detects changes & reloads browser as you work

Using

You can configure what fonts,js & css are imported automatically from node modules by editing the top of the gulpfile.js

Then build your project in the src folder, keeping css/scss in the scss file & js in the js folder, images in the img folder, and any fonts you want to use in the fonts folder

You can use partials & auto-inject js/css in your html files (see index.html for example)

Your files will automatically be copied to the app folder

Then use the following commands:

  • npm start: Run a build server that reloads browser as changes are made
  • npm run build: Creates a production build that uses minified JS/CSS
  • npm test:Test the build is correct

In case of error like css files not being injected, run the command a second time
